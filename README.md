# multi-purpose-lab

A starter repository for building multipurpose apps, libraries, and data science projects using Python, JavaScript, Node.js, and React.

## Structure

- **src/backend/**: Python backend (API/data scripts)
- **src/frontend/**: React app (JavaScript/Node.js)
- **data/**: Data files/resources
- **docs/**: Project documentation

## Getting Started

Clone the repo and check the `docs/index.md` for setup instructions.
